# node-red-contrib-memwatch

Node Red 调试工具箱,用于监控内存相关信息


## 安装

**Requires Node.js v8.0 or newer**

Either install from the Node-RED palette manager, or:

```
$ npm i node-red-contrib-memwatch
```
